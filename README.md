# TCC - Solução IoT para Monitoramento Industrial

Este repositório apresenta a estrutura conceitual de uma solução baseada em Internet das Coisas (IoT) para monitoramento em tempo real de máquinas industriais em uma metalúrgica.

## Objetivo

Propor uma arquitetura capaz de coletar, transmitir, armazenar e visualizar dados operacionais de máquinas industriais, contribuindo para o acompanhamento dos processos produtivos e para a tomada de decisão baseada em dados.

## Tecnologias previstas

- C#
- ASP.NET
- SQL Server
- HTML
- CSS
- JavaScript
- Dispositivos IoT
- Sensores industriais

## Telas previstas

O protótipo contempla as seguintes telas:

- Tela de login
- Dashboard de monitoramento em tempo real
- Tela de detalhes da máquina
- Tela de alertas operacionais

## Funcionamento conceitual

A solução foi planejada para receber dados simulados de sensores instalados em máquinas industriais. Esses dados seriam enviados para uma aplicação web, armazenados em banco de dados e exibidos em dashboards para acompanhamento em tempo real.
